# AIStream-IBMi


Usage notes:

```bash
java -jar aistream.jar [[option] ... ]
   Options include:
     --action <LIST/ADD/GET/REMOVE>   The action you are performing
     --table <tablename>              The table name of interest (required for ADD/GET/REMOVE actions)
     --schema <schemaname>            The schema name of interest (required for ADD/GET/REMOVE actions)
```
